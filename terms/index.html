About Spokn

  • Spokn is a personal dictation utility for iPhone and Mac.
  • It provides two transcription engines: an on-device engine (using WhisperKit) and an online engine that calls OpenAI's transcription API using your own API key.
  • There is no SpoknLabs account, no in-app purchase, and no subscription.
  • SpoknLabs operates no backend service — Spokn runs entirely on your device, except for transcription requests you choose to route through OpenAI.
