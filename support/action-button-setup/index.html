Set up the Action Button for Spokn

On iPhone 15 Pro and newer, you can start and stop Spokn with a single press of the Action Button — the small button just above the volume buttons. The whole setup takes about a minute.

Before you start

  • You need an iPhone with an Action Button — that's iPhone 15 Pro, iPhone 15 Pro Max, and any newer Pro / Pro Max model.
  • Spokn must be installed and opened at least once. The first launch is what registers Spokn's shortcut with iOS.
  • Spokn needs microphone permission. If you skipped that prompt, you can enable it later in iPhone Settings → Spokn → Microphone.

Install the Spokn shortcut

You don't have to install anything by hand. Once Spokn is on your phone and you've opened it once, iOS automatically picks up the Spokn shortcut and makes it available everywhere shortcuts can run — including the Action Button.

To confirm it's there:

  1. Open the Shortcuts app on your iPhone.
  2. Tap App Shortcuts at the top.
  3. Look for the Spokn section. You should see an entry called Spokn — Start Dictation (or similar).

If you don't see it, open Spokn once and try again — iOS sometimes needs a moment to register newly installed app shortcuts.

Assign it to the Action Button

  1. Open iPhone Settings.
  2. Tap Action Button.
  3. Swipe through the row of modes (Silent, Focus, Camera, Flashlight, etc.) until you reach Shortcut.
  4. Tap Choose a Shortcut.
  5. Pick Spokn — Start Dictation from the list. You can search for "Spokn" if it's not visible right away.
  6. Leave Settings. The choice is saved automatically.

Use it

  • Press the Action Button to start a dictation session.
  • Press it again to stop. Spokn transcribes what you said and copies the text to your clipboard.
  • The first time you do this, iOS may ask you to confirm running the shortcut. Tap Always Allow so it doesn't ask again.

Troubleshooting: Action Button opens Spokn but doesn't record

If pressing the button brings up Spokn but no recording starts, run through these checks in order:

  1. Microphone permission. Open iPhone Settings → Spokn → Microphone and make sure it's on. Without this, Spokn can launch but cannot capture audio.
  2. Right shortcut selected. Open iPhone Settings → Action Button → Shortcut and confirm it shows Spokn — Start Dictation and not a different shortcut.
  3. "Always Allow" the shortcut. If iOS is still showing a confirmation prompt each time, tap Always Allow when it appears so the shortcut runs without interruption.
  4. Open Spokn once, manually. If you've just installed Spokn or recently updated it, launching the app once helps iOS finish registering the shortcut.
  5. Restart the phone. Rare, but iOS occasionally loses track of an Action Button binding. A reboot reliably fixes that.
  6. Re-pick the shortcut. In Settings → Action Button, switch to a different mode, then back to Shortcut and re-select Spokn — Start Dictation.

Still stuck? Email spoknlabs@gmail.com with your iPhone model, iOS version, and Spokn version (visible in Spokn → Settings). It makes the fix much faster.
