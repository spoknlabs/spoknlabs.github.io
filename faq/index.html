About Spokn

Spokn

Does Spokn work offline?
Yes. Set the transcription mode to Local only (or leave it on Auto, which uses the on-device engine when you have no network). Audio never leaves your iPhone in either of those paths.
Do I need an OpenAI API key?
Only if you want to use the OpenAI engine. In Local only mode, Spokn uses an on-device model and needs no key. If you're in Auto or Online only mode, you'll add a key in Settings, and it's used directly against OpenAI's API.
Where is my API key stored?
In the iOS Keychain, on your device. Spokn uses it as the Bearer token on transcription requests and never transmits or inspects it otherwise.
Does Spokn record or save my audio?
Audio is written to a temporary file only while recording, then deleted the moment the transcript is delivered (or the attempt is abandoned). Transcripts live in memory for the session. There is no server and no database. See the Privacy Policy for specifics.
Is there a subscription?
No. Spokn is free. If you use the OpenAI engine, your usage is billed directly to your own OpenAI account — we don't see or touch that.
Can I use Spokn for professional work?
It's fine for everyday use — meeting notes, emails, code comments, captions. We don't recommend relying on it for legal, medical, or safety-critical transcription. See the Terms of Service.
How do I uninstall Spokn and remove all data?
Delete the app from your iPhone. iOS removes the app's Keychain entries, UserDefaults, and any downloaded on-device model along with it. There's no server-side account to delete.

Who makes SpoknLabs apps?
SpoknLabs is a small indie studio. We build native apps for Apple platforms and ship on our own schedule.
Do you run a mailing list?
No regular newsletter. If you email us asking to be notified about a specific app, we'll keep you on a short list and send a single note when it's out.
Are your apps open source?
Not currently. Reach out if you have a specific interest — we're open to conversations.
