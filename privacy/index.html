Privacy Policy

Last updated: April 24, 2026

This policy covers Spokn, the iOS dictation app from SpoknLabs. Other SpoknLabs apps, if any, have their own privacy pages.

What Spokn processes

  • Audio recordings captured via the microphone when you dictate.
  • Transcription text returned from the configured engine.
  • Vocabulary hint — an optional text field of domain terms you save in Settings. It's passed to the transcription engine as a prompt to improve accuracy.
  • Language preference — an optional ISO-639-1 code you pick in Settings.

Where processing happens

Spokn v3+ runs in one of three modes, user-selectable in Settings:

  • Auto (default). Transcribe with OpenAI when the device is online; on-device (WhisperKit) when offline.
  • Online only. Always transcribe with OpenAI.
  • Local only. Always transcribe on-device. Audio never leaves your iPhone.

When a recording is transcribed via OpenAI, the audio and any vocabulary hint / language fields are sent to OpenAI's /v1/audio/transcriptions API using your own API key. When a recording is transcribed locally, nothing leaves the device.

Purpose

  • Provide the core dictation service — speech to clipboard text.
  • No analytics, telemetry, or marketing processing.

Retention & deletion

  • Audio files are temporary — written to the app's temp directory during recording and deleted immediately after the transcript is delivered or after a failed attempt is abandoned.
  • Transcripts are held in memory only for the duration of the app session plus the most recent one (shown in the in-app "Copy again" card). They are not archived, synced, or written to persistent storage.
  • No user accounts. No server-side records. Spokn has no backend.
  • Your OpenAI API key is stored locally in the iOS Keychain. Spokn does not transmit or inspect it beyond using it as the Bearer token in the transcription request.

Third-party processors

  • OpenAI — only when Online mode is active or Auto mode routes online. Audio and text fields are sent as documented in OpenAI's API reference. Retention and use are governed by OpenAI's own API data policies (API data is not used to train models by default).

Your rights

  • Because Spokn holds no server-side data, there's no account to delete. Uninstalling the app removes all local state — Keychain entries, UserDefaults, and any downloaded on-device model.
  • To revoke the OpenAI path, delete your API key in Settings. To revoke the on-device path, delete any downloaded model in Settings.

Contact

Questions about privacy? Email hello@spoknlabs.com. See the Support page for more ways to get in touch.
